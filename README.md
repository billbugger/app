Install
-------

```bash
composer create-project fr3nch13/billbugger html
cd html
git init
git add .
git commit -m "Initial commit"
./bin/build app:install DB_NAME=my_app,PROJECT_NAME="My Project",PROJECT_VERSION="v1.0.0"
```

Update
------

```
composer update
```
